# AlpacaCraft

AlpacaCraft is a fully responsive mock e‑commerce experience showcasing handcrafted alpaca wool goods through a modern, accessible, and interactive front‑end. The site dynamically loads product data from JSON, renders a responsive product grid, and provides an elegant modal system for viewing item details. It also includes smooth scrolling, a theme toggle, a cookie‑consent banner, and a polished contact form with client‑side validation — all built with clean, framework‑free JavaScript. The project is deployed using AWS Amplify to provide fast, reliable hosting and streamlined CI/CD integration.

The codebase emphasizes thoughtful UI/UX design, semantic HTML, and CSS architecture that supports both light and dark themes. Accessibility considerations such as keyboard navigation, ARIA attributes, and focus management are integrated throughout. The project also incorporates basic SEO improvements through descriptive metadata, structured headings, and clean URL patterns. AlpacaCraft serves as a compact demonstration of front‑end engineering fundamentals, modular JavaScript patterns, and user‑centric design.
